*This EIP allows EOAs to execute contract code by implementing a new transaction type, tx type "4"*

### Mental model

This EIP basically introduces "delegateCall" but for EOAs - it allows EOAs to operate contract code with their own storage context
- important to note that EOAs don't have "storage" per say, but they have nonces and balances
